# Modern Calculator App

A beautiful, feature-rich calculator web application with a glassmorphism UI design.

**Live Demo:** [https://evancoder36.github.io/calculatorapp/](https://evancoder36.github.io/calculatorapp/)

## Features

### Basic Calculator
- Standard arithmetic operations (+, -, *, /)
- Percentage calculations
- Keyboard support for quick input

### Scientific Calculator
- Trigonometric functions (sin, cos, tan, and inverses)
- Logarithmic functions (log, ln, log2)
- Powers and roots (x², x³, xʸ, √x, ³√x)
- Constants (π, e)
- Factorial and absolute value
- Memory functions (M+, M-, MR, MC)
- RAD/DEG mode toggle

### Graph Plotting
- Plot mathematical functions (e.g., `sin(x)`, `x^2 + 2*x - 1`)
- Interactive zoom and pan
- Multiple functions with different colors
- Customizable axis range
- Grid toggle

### Unit Converter
- **Length:** meters, km, miles, feet, inches, etc.
- **Weight:** kg, grams, pounds, ounces, etc.
- **Temperature:** Celsius, Fahrenheit, Kelvin
- **Volume:** liters, gallons, cups, etc.
- **Area:** m², km², acres, hectares, etc.
- **Speed:** m/s, km/h, mph, knots
- **Time:** seconds to years
- **Data:** bits to petabytes

### Currency Converter
- Live exchange rates from ExchangeRate-API
- 30+ currencies with flags
- Quick-select popular currencies
- Swap button for easy conversion
- Offline fallback rates

### Calculation History
- Persistent history using localStorage
- Click to reuse previous calculations
- Clear all history option

## Tech Stack

- **HTML5** - Semantic structure
- **CSS3** - Glassmorphism design with animations
- **JavaScript** - Vanilla JS, no frameworks
- **Math.js** - Advanced mathematical operations
- **Plotly.js** - Interactive graph plotting
- **ExchangeRate-API** - Live currency rates

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/evancoder36/calculatorapp.git
   ```

2. Open `index.html` in your browser

Or simply visit the [live demo](https://evancoder36.github.io/calculatorapp/).
